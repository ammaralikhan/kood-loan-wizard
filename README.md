1. Download or clone this repo
2. Open `index.html` directly in Chrome or Firefox
3. No build step required — works out of the box

## 🧮 Loan Calculator Formula
Monthly payment calculated using the standard amortization formula:

M = P * [r(1+r)^n] / [(1+r)^n - 1]

Where:
- `M` = Monthly payment
- `P` = Principal (loan amount)
- `r` = Monthly interest rate (annual rate ÷ 12 ÷ 100)
- `n` = Number of months

Source: [Wikipedia — Amortization Formula](https://en.wikipedia.org/wiki/Amortization_formula)
